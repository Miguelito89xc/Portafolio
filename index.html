Sobre Mi

Soy un entusiasta programador en mis primeros pasos en el mundo del desarrollo web. Actualmente, mi enfoque se centra en dominar las bases fundamentales de HTML, CSS y Bootstrap. Estoy fascinado por la capacidad de estructurar páginas web con HTML, aplicar estilos visualmente atractivos utilizando CSS, y aprovechar la potencia y facilidad de uso de Bootstrap para crear diseños responsivos y funcionales.

Mis Proyectos

Proyecto 1

He creado una página web de zapatos utilizando HTML, CSS y Bootstrap para el diseño responsivo. La estructura incluye un encabezado con barra de navegación, s secciones para mostrar productos, detalles de producto con imágenes y descripciones, formulario de contacto y un pie de página con información adicional y enlaces a redes sociales

Proyecto 2

He diseñado una página web de maquillaje utilizando HTML, CSS y Bootstrap para garantizar un diseño responsivo y atractivo. La página incluye un encabezado con barra de navegación, secciones dedicadas a diferentes tipos de productos de maquillaje como base, labiales, y sombras de ojos, cada una con imágenes y descripciones detalladas. Además, cuenta con testimonios de clientes, formulario de contacto y un pie de página con información adicional y enlaces útiles.

Proyecto 3

Estructuré el contenido, diseñé el aspecto visual con colores cálidos y dorados, integré imágenes de productos y funcionalidades como formulario de contacto y mapas interactivos, garantizando una experiencia de usuario fluida y atractiva en diversos dispositivos.
